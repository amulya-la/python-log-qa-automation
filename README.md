# Python File-Handling and QA Automation Script
Thhis is Python automation script designed to validate QA log files based on naming conventions, date formats, and file extencions.
## Features
- **File Segregation:** Automatically validates log files and separates them into invalid folders.
- **Robust Error Handling:** Uses Exception HAndling to prevent crashes if a file has an incorrect format or invalid date.
- **Execution Logs:** Generates an Execution Log file ('automation.py') to keep track of the process for audit and traceability.

## Tech Stack
-Python (Modules): 'os', 'shutil', 'logging', 'datetime')
